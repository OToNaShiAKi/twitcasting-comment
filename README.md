# twitcasting-comment

## Project description

TwitCasting 和 Bilibili 弹幕链接翻译工具。
可以显示消息、礼物、醒目留言、表情、会员、消息翻译，界面美观易读。
可设置主题色、显示内容、界面语言

## Project setup

```
yarn install
```
